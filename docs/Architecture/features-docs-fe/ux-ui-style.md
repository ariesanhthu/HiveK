# 1. Recommended Brand Palette (Final)

### Primary

**Amber / Warm Orange**

```
#F59E0B
```

Ý nghĩa:

* năng lượng
* creator economy
* không quá "bee yellow"

Use cho:

* primary button
* highlight
* chart accent

---

### Secondary

**Modern Orange**

```
#FB923C
```

Use cho:

* CTA
* campaign highlight
* hover states

---

### Accent (Tech Blue)

```
#3B82F6
```

Use cho:

* charts
* analytics
* links

Blue giúp UI **tech hơn và cân bằng màu nóng**.

---

### Accent (Creator Purple)

```
#8B5CF6
```

Use cho:

* creator stats
* badges
* ranking highlights
